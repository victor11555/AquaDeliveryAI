## Когда использовать

Открывай нужный файл, когда задача требует понимания:
- как устроен интерфейс конкретного продукта (разделы, формы, поля, кнопки);
- бизнес-логики и процессов Aqua Delivery;
- архитектуры, БД, интеграций и ограничений системы.

Это справочные материалы — не алгоритмы и не скиллы, обновляются по мере изменений продуктов.
