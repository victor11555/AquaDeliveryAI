> **Версия:** 4.5.15
> **App Store:** https://apps.apple.com/ru/app/клиент-aqua-delivery/id1451225794
> **Разработчик приложения:** Мобильные Решения ООО (рейтинг 5.0★, бесплатно)

## Как устроено приложение — общая логика

```
Клиент открывает сайт/приложение
  → видит Главную (баннеры + каталог товаров)
  → выбирает товар → Карточка товара → добавляет в Корзину
  → Корзина → вводит промокод, считает возвратную тару
  → Оформление заказа (адрес, дата, интервал, способ оплаты)
  → Заказ создан

Параллельно доступно:
  → Личный кабинет (профиль, адреса, баллы, история, чат)
  → Панель баллов (история начислений, правила)
  → Чат с оператором (поддержка)
```

**Ключевые принципы:**
- Адрес доставки выбирается в шапке — влияет на каталог, цены и доступные интервалы.
- Бонусные баллы: 1 балл = 1 рубль; начисляется +5% от каждой покупки после успешной доставки; за отменённые не начисляются.
- Возвратная тара управляется прямо в корзине через счётчик «Сдаю пустых бутылей».
- Телефон и email изменить самостоятельно нельзя — только через оператора.

## 1. Шапка (Header)

**Присутствует на всех страницах.** На странице `/checkout` упрощена: только «‹ Назад», название и иконки баллов и профиля.

| Элемент | Где | Что делает / куда ведёт |
|---|---|---|
| Логотип + название партнёра | Левый верхний угол | Клик → главная страница (`/`) |
| Подзаголовок «Доставка чистой воды» | Под названием | Информационный, не кликабелен |
| Иконка геолокации + адрес доставки | Центр шапки | Клик → модальное окно «Выберите адрес» (`/addresses`) |
| Шеврон ▼ рядом с адресом | Рядом с адресом | Тоже открывает окно выбора адреса |
| Поле «Поиск товара» | Справа от адреса | Активирует поиск по каталогу в реальном времени |
| Иконка баллов (градиентный кружок) | Правый верхний угол | Клик → панель «Мои баллы» (`/bonus`) |
| Иконка корзины (синяя) | Правый верхний угол | Клик → панель «Корзина» (`/cart`) |
| Иконка профиля (силуэт человека) | Правый верхний угол | Клик → «Личный кабинет» (`/profile`) |
| «‹ Назад» (только на `/checkout`) | Верхний левый угол | Возврат к корзине (`/cart`) |

## 2. Главная страница (`/`)

| «БИЗНЕС В ЦИФРЕ» | Внешняя ссылка | Внешний ресурс / материал |
| НАЧАЛО ДРУЖБЫ 1190Р (было 1590Р) | Акционный | Переход к карточке акции в каталоге |
| Капля (синий брендовый) | Брендовый | Полноэкранная сторис |
| «ПОЧЕМУ ВАМ больше не надо ждать доставку» | Информационный | Сторис с описанием преимущества |

| Промокод (пример: ZZZUG0) | Промокод | Сторис с промокодом и условиями акции |

| Элемент | Действие |
|---|---|

## 3. Карточка товара (`/products/{id}`)

Открывается кликом на карточку в каталоге. Отображается как **модальное окно поверх страницы**.

### 3.1 Обычный товар

| Элемент | Описание / действие |
|---|---|
| Фото товара + стрелки ‹ › | Галерея. Стрелки и точки-индикаторы внизу переключают фото |
| Название товара | Полное название (пример: «Вода 19л "Ваш Бренд" 777») |
| Блок цен «от N шт» | Ценовые уровни: от 3шт, от 4шт, от 5шт, от 7шт — пересчитываются при добавлении в корзину |
| «бонусных баллов: +N б» | Кешбэк за 1 шт. Показывается под ценами |
| Описание товара (прокручиваемое) | Полный текст описания |
| Блок «Что ещё пригодится» | Сопутствующие товары с кнопками добавления в корзину |
| Кнопка «N Р / шт.» + «В корзину» | Добавляет 1 единицу в корзину |
| Кнопка «×» | Закрывает карточку, возврат к каталогу |

### 3.2 Акционный товар

| Элемент | Описание / действие |
|---|---|
| Крупное изображение акции | Фото на весь блок |
| Название акции | Пример: «Две бутыли по цене одной» |
| «бонусных баллов: +N б» | Кешбэк за набор |
| «Остаток: N» | Количество доступных единиц на складе (пример: «Остаток: 98769») |
| Описание условия | Текст акции (пример: «При покупке одной бутыли — вторая в подарок!») |
| Кнопка «N Р / шт.» + «В корзину» | Добавляет набор в корзину |
| Кнопка «×» | Закрывает карточку |

## 4. Корзина (`/cart`)

**Боковая панель справа.** Открывается кликом на иконку корзины в шапке.

### 4.1 Пустая корзина

| Элемент | Действие |
|---|---|
| «Добавьте товары в корзину» | Сообщение-заглушка |
| Кнопка «К списку товаров» | Закрывает корзину → возврат к главной (`/`) |
| «Сдаю пустых бутылей» − 0 + | Счётчик возвратной тары — доступен даже в пустой корзине |
| Кнопка «Оформить заказ 0 Р» | Неактивна (серая) пока корзина пуста |

### 4.2 Корзина с товарами

| Элемент | Действие |
|---|---|
| Фото + название + «N Р за 1 шт.» | Информация о позиции |
| «Сумма: N Р» | Итог по позиции |
| Кнопка «×» рядом с товаром | Удалить позицию из корзины |
| Счётчик «− N +» | Изменить количество единиц |
| Блок «С этим товаром обычно берут» | Сопутствующие товары — можно добавить не закрывая корзину |
| Поле «Промокод» / «Введите промокод» | Ввод промокода; скидка применяется сразу после ввода |
| «Сдаю пустых бутылей» ⓘ − N + | Возвратная тара; ⓘ — подсказка о залоге за тару |
| «Доставка — N Р» | Стоимость доставки (рассчитывается автоматически по зоне) |
| Кнопка «Оформить заказ N Р» | Итоговая сумма. Клик → страница оформления (`/checkout`) |
| Кнопка «×» | Закрыть корзину без потери состава |

## 5. Оформление заказа (`/checkout`)

**Полная страница** (не боковая панель). Шапка упрощена: «‹ Назад», название, адрес, иконки баллов и профиля.

### 5.1 Способ получения

| Вкладка | Описание |
|---|---|
| Доставка (по умолчанию) | Курьер доставляет по указанному адресу |
| Самовывоз | Клиент забирает заказ из офиса продаж партнёра |

### 5.2 Адрес доставки

| Поле | Описание |
|---|---|
| Адрес (автозаполнение) | Подставляется выбранный адрес из профиля (пример: «Куратова, д. 91, Сыктывкар») |
| Подъезд | Необязательное |
| Этаж | Необязательное |
| Кв./Офис | Необязательное |
| Комментарий к адресу | Домофон, ориентир — пояснения курьеру |
| «+ Указать другой адрес» | Ввод нового адреса с геоподсказками прямо на этой странице |

### 5.3 Дата и время доставки

| Элемент | Описание |
|---|---|
| Сегодня | Доставка в текущий день |
| Завтра | Доставка на следующий день |
| Иконка календаря | Датапикер — выбрать любую доступную дату |
| Интервалы времени (пример: 15:15–20:15) | Доступные слоты для выбранной даты и зоны. Клик → выбор |

### 5.4 Бонусные баллы

| Элемент | Описание |
|---|---|
| Блок «Бонусные баллы» (градиентная иконка) | Показывает текущий баланс. Клик — применить / снять баллы в счёт оплаты |

### 5.5 Способы оплаты (8 вариантов)

| Способ | Описание |
|---|---|
| QR code | Оплата QR-кодом при доставке — курьер показывает, клиент сканирует через банковское приложение |
| Юмани карта с привязкой | Оплата через ЮMoney привязанной картой без повторного ввода данных |
| СБП | Система быстрых платежей — перевод по QR через любое банковское приложение |
| Проверочный | Тестовый способ (только в тестовой среде) |
| По счёту (для юр. лиц) | Безналичная оплата по счёту — доступна при заказе от организации |
| СБП с привязкой | СБП с привязанным аккаунтом — без повторного сканирования QR |
| Юмани без привязки | Оплата через ЮMoney разовым платежом без привязки карты |
| Наличка | Оплата наличными курьеру при доставке |

### 5.6 Дополнительные параметры

| Поле / чекбокс | Описание |
|---|---|
| Комментарий к заказу | Любые пожелания курьеру или оператору |
| ☑ «Не звонить мне» (по умолчанию включён) | Курьер не будет звонить перед доставкой |
| ☐ «Оставить заказ у двери» ⓘ | Бесконтактная доставка; ⓘ — всплывающее пояснение условий |

### 5.7 Блок «Заказ и стоимость»

| Элемент | Описание |
|---|---|
| Список товаров с количеством | Название + кол-во (пример: 1 шт.) |
| Сумма заказа | Стоимость товаров |
| Доставка | Стоимость доставки |
| К оплате | Итого |
| Кнопка «Оформить заказ» | Создаёт заказ → переход к подтверждению / оплате |

## 9. Панель «Мои баллы» (`/bonus`)

**Боковая панель.** Открывается иконкой баллов в шапке или из Личного кабинета.

### 9.1 Вкладка «История»

| Элемент | Описание |
|---|---|
| «N баллов» (заголовок) | Текущий баланс клиента |

### 9.2 Вкладка «Правила использования»

| Правило | Суть |
|---|---|
| 1 балл = 1 рубль | Курс конвертации при оплате |
| Как получить? | +5% от каждой покупки. Начисляются после успешной доставки. За отменённые — не начисляются |
| Как потратить? | Оплата покупок баллами при оформлении через приложение и сайт |

## 10. Управление адресами (`/addresses`)

**Модальное окно.** Открывается кликом на адрес в шапке или «Мои адреса» в профиле.

| Элемент | Действие |
|---|---|
| Кнопка «+» | Добавить новый адрес — форма с геоподсказками |
| Список адресов с радиокнопкой ○/● | Выбор активного адреса; ● синяя галочка = текущий |
| Иконка корзины (красная) | Удалить адрес → диалог «Вы уверены? / Нет / Да» |
| Кнопка «×» | Закрыть без изменений |

> **Важно:** выбранный адрес влияет на каталог, доступные интервалы доставки и цены.

## 11. О компании (`/about`)

**Боковая панель.** Открывается из профиля → «О компании» или кнопкой «О компании» в разделе Комбо.

| Элемент | Тип / действие |
|---|---|
| Название партнёра | Заголовок (информационный) |
| Адрес офиса | Текст (пример: «Иванова 113») |
| Телефон (с шевроном ∨) | Раскрывающийся блок; клик по номеру → звонок (`tel:`) |
| Email (с шевроном ∨) | Раскрывающийся блок; клик → почтовый клиент (`mailto:`) |
| Сайт (ссылка) | Прямая ссылка → сайт партнёра в новой вкладке |
| «Реквизиты юридического лица» (∨) | Раскрывается: ИНН, КПП, ОГРН, банковские реквизиты |
| «Документы» (∨) | Раскрывается: лицензии, договоры, документы |
| Текст о компании | Описание деятельности партнёра |
| «Сертификаты» + изображения | Сканы сертификатов качества (пример: «Чистая вода 100%»); если офисов несколько — каждый офис показан отдельным блоком |
| Несколько офисов | Каждый офис — отдельный блок с адресом, телефоном, email, сайтом |
| Кнопка «×» | Закрывает панель |
