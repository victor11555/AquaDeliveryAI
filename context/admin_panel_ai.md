### Общие паттерны интерфейса

- **Хлебные крошки** — всегда показывают путь, последний элемент = текущая страница.
- **Фильтры** — строка инпутов под заголовками таблицы; применяются по Enter или автоматически.
- **Сортировка** — клик по заголовку колонки меняет направление сортировки.
- **Языковые вкладки** (Узбекский / Русский / Azerbaijani) — встречаются на большинстве страниц редактирования; каждая вкладка содержит отдельные текстовые поля для своей локали.
- **Мультиселект** — поле с поиском; удаление элемента через «×» рядом с ним.
- **Поле даты** — иконка календаря открывает пикер; иконка «×» очищает фильтр.
- **Подвал** — «Aqua Delivery 2026» + версия релиза (клик — история обновлений).

## 2. Раздел «Товары»

### Категории товаров
**URL:** `https://admin.test.aqua-delivery.ru/category/`

**Кнопка «Создать»** — форма: название, последовательность, видимость, флаг «по акции», привязка к родителю.

**Таблица:**

| Колонка | Поведение |
|---|---|
| Название | Ссылка на форму редактирования категории |
| Товары в категории | Ссылка «Товары» → список товаров по этой категории; строка «Комбо» → product-set |
| Последовательность | Нажатие на число → изменение порядка |
| Отображается | Иконка вкл/выкл; клик → меню редактирования |
| По акции | Иконка; клик → меню редактирования |
| Действия | ✏ редактирование; корзина — удаление с подтверждением (некоторые категории защищены) |

**Фильтры:** ID, Название, Последовательность, Отображается (Вкл/Выкл/все).

---

### Массовое изменение цен товаров
**URL:** `https://admin.test.aqua-delivery.ru/product-price/change-product-price/`

Пакетное изменение цен через Excel.

1. «Экспорт в Excel» — скачать актуальный прайс.
2. Изменить цены в файле.
3. Сохранить файл.
4. Загрузить в блоке «Импорт товаров из Excel».
5. «Сохранить» — применить изменения.

---

### Наборы товаров
**URL:** `https://admin.test.aqua-delivery.ru/product-set/`

**Кнопка «Создать»** — форма: название, последовательность, флаги «Показывать в акциях/баннерах/товарах», период акции, лимит заказов.

**Таблица — ключевые колонки:**

| Колонка | Описание |
|---|---|
| Название | Ссылка на редактирование набора |
| Показывать в акциях / баннерах / товарах | Иконки вкл/выкл |
| Начало / Конец акции | Даты промо-периода |
| Количество заказов | Лимит — при достижении акция автоматически прекращается |
| Действия | «Изменить», удаление с подтверждением |

---

### Редактирование набора товаров
**URL:** `https://admin.test.aqua-delivery.ru/product-set/update/?id=ID`

**Языковые вкладки:** Название и Описание отдельно на каждом языке.

**Чекбоксы:**
- Главное изображение, Показывать в баннерах, Показывать в акциях, Доступен при экспрессе, Показывать в товарах.

**Период и лимиты:**
- Начало/Конец акции (с календарём).
- «Количество заказов» — если указано 1, клиент с 1+ завершёнными заказами набор не видит.
- Минимальное/Максимальное количество в заказе.

**Зоны доставки:** мультиселект — добавить/удалить (×).

**Состав набора:** таблица Товар (#id) / Количество / Цена / удаление (×). Кнопка «+» — добавить товар.

**Служебные поля (readonly):** «Вознаграждение(%)», «Создано в», «Изменено в».

### Редактирование товара
**URL:** `https://admin.test.aqua-delivery.ru/product/update/?id=ID`

Ссылка «Изменить цены» → `/product/update-prices/?id=` (цены по зонам).

**Основные поля:**

| Поле | Примечание |
|---|---|
| Название | Обязательное |
| Описание | Текст для сайта/приложения |
| Объем | Литры или иные единицы |
| Отображается | Чекбокс видимости |
| Доступен при экспрессе | Чекбокс |
| Включить маркировку | Чекбокс учёта кодов DataMatrix |
| Не валидировать маркировку (для КИТУ) | Чекбокс — отключает стандартную проверку |
| GTIN | Основной код товара |
| GTIN дополнительные | Через «+» |
| Базовый тип | Продукт / Услуга |
| Категория | Акции, Наборы, Вода, Оборудование, Штучные товары, Услуги, Напитки, Склад и др. |
| Тип товара | Вода 19л, Оборудование, Напитки и др. (влияет на правила цены и доставки) |
| Мин./Макс. количество в заказе | Ограничения |
| Условные единицы загрузки | 1 ед. = 1 бутыль 19л; дробные допускаются |
| Сопутствующие товары | Поиск и добавление (апселл/кросс-селл) |
| Отображать в зонах доставки | Мультиселект зон |
| Цена | Обязательное; базовая цена |
| Вознаграждение(%) | Комиссия для агентов |
| Базовый товар | Если выбран — при добавлении в корзину заменяется на него в указанном кол-ве |

**Изображения:**
- Главное (1:1, макс. 1 МБ) — при наличии дополнительных НЕ показывается при нажатии.
- Дополнительные: «+» новый слот, «Close» удаление, бургер-меню = порядок.
