Присутствует на всех страницах CRM. Переключение разделов без перезагрузки.

| 🔔 | Оповещения | Красный бейдж = непрочитанные |
| 💧 | Заказы | Активный раздел подсвечен синим |
| 📋 | Маршрутные листы | — |
| 👥 | Клиенты | — |
| 🏢 | Организации | — |
| 📍 | Адреса | — |
| 🚗 | Водители | — |
| 💬 | Чат | Красный бейдж = непрочитанные |

| 🏪 | МикроМаркеты | — |
| 🚪 | Выход | — |

### Настройка звука уведомлений

Можно включить звук при: Новом заказе / Чат / Раздел оповещений.

### Раздел оповещений

**Шапка:**
- 💬 — пометить все прочитанными.
- 🔽 фильтр → 3 инпута:

  - Период: Сегодня / Завтра / Вчера / Последние 7–30 дней / Этот месяц / Свой диапазон.

**Список:** время (ДД.ММ.ГГГГ / ЧЧ.ММ) + текст. Синие ссылки в тексте → переход к заказу или клиенту.

## 2. Заказы — Табличный вид

**URL:** `https://v.test.aqua-delivery.ru/orders`

### Шапка

| Элемент | Действие |
|---|---|
| ❓ | Справка: как создать / редактировать заказ |
| «Скачать ▾» | Экспорт в CSV: заказы и/или товары, можно выбрать отдельные поля или все |
| «+ Новый заказ» | Открыть форму создания заказа |

### Фильтры

| Элемент | Действие |
|---|---|
| «Тип заказа ▾» | Экспресс / Новый клиент / Просроченный |
| «Все зоны доставки ▾» | Фильтр по зоне |
| «Все колонки ▾» | Настройка видимых колонок |
| «Сортировка ▾» | Поле + порядок (ФИО / доставка / водитель / дата создания) |
| Тогглер «Показать на карте» | Переключение таблица ↔ карта |
| Иконка три точки | Переход: планировщик заказов, пропущенные звонки, возврат клиентов, кешбэк, отчёт за день, онлайн-оплата |

### Колонки таблицы

| Колонка | Тип | Действие |
|---|---|---|
| ☑ Чекбокс | — | Выбрать / снять все |
| № + поле | Поиск | Поиск по номеру заказа |
| 📦 Иконка тары | Фильтр | По типу создания заказа |
| ФИО + тел + email + поле | Поиск | По клиенту |
| Адрес + поле | Поиск | По адресу доставки |
| Район ▾ | Фильтр | По району |
| Дата доставки 📅 | Фильтр | Календарь: сегодня / завтра / вчера / последние 7/30 дней / этот месяц / свой диапазон |
| Интервал ▾ | Фильтр | Временной интервал доставки |
| Состав ▾ | Фильтр | По товарам в заказе |
| Водитель ▾ | Фильтр | По назначенному водителю |
| Иконка кошелька | Фильтр | Все / оплаченные / не оплаченные / наличные / безналичный по договору / картой курьеру / онлайн-оплата / курьеру по телефону / курьеру по QR-коду / купонами / ERIP |
| Ст | Сортировка | По статусу заказа |
| Дата создания 📅 | Фильтр | По дате создания |

### Действия в строках

| Элемент | Действие |
|---|---|
| ☑ Чекбокс | Выбрать для массового действия |
| Номер заказа (синий) / «Экспресс» | Открыть карточку заказа |
| ФИО клиента | Перейти в карточку клиента (при наведении — номер телефона) |
| 🧴 Состав + цифра | Просмотреть состав заказа |
| Водитель ▾ | Выпадающий список — назначить водителя |
| Статусный бейдж | Изменить статус заказа |

### Статусы заказов

| Бейдж | Цвет | Значение |
|---|---|---|
| Тп | Красный | Требует подтверждения |
| Н | Оранжевый | Новый |
| П | Фиолетовый | Принят |
| В | Синий | Выполняется |
| Д | Зелёный | Доставлен |
| О | Малиновый | Отменён |
| Нд | Серый | Не доставлен |

### Массовые действия

После выбора чекбоксов:
- Назначить водителя нескольким заказам.
- Просмотреть состав заказов.

### Пагинация и параметры URL

| Параметр | Назначение |
|---|---|
| `v_page=` | Номер страницы |
| `v_per-page=` | Записей на странице (20 / 50 / 100) |
| `f_orderProductType=` | Состав заказа |
| `f_shippingAndDeliveryDateTime=` | Дата доставки |
| `v_orderAddressFulltext=` | Адрес доставки |
| `v_clientFulltext=` | ФИО / тел. / email клиента |
| `v_orderAddressFulltext=` | Адрес доставки (дублирует, см. выше) |
| `f_id=` | Номер заказа |

### Типы создания заказа

Оператор / Приложение / Сайт / Обмен / ММ-сайт / ММ-приложение (в исходнике: «ММ-приложние»).

## 4. Карточка заказа

**URL:** `https://v.test.aqua-delivery.ru/orders/:id`

### Навигация и заголовок

- «‹ К списку заказов» — вернуться без потери фильтров.
- «Заказ #XXXXX» — номер заказа; иконка рядом — способ создания.

### Статус и оплата

- «🟠 Новый ▾» — выпадающий список смены статуса.
- «По счёту (для юр. лиц) ▾» — выпадающий список смены способа оплаты.

### Кнопки действий

| Кнопка | Действие |
|---|---|
| «Сформировать ссылку на оплату» | Генерировать платёжную ссылку; тост об ошибке при неудаче |
| «Создать заказ» | Дублировать текущий заказ |

### Блок комментария оператора

- «+» → добавить комментарий.
- Редактирование: удалить текст → комментарий удаляется.

### Блок клиента

| Элемент | Действие |
|---|---|
| ФИО клиента (синяя ссылка) | Открыть карточку клиента |
| Телефон | Номер телефона |
| 🚫 рядом с телефоном | Заблокировать клиента / чёрный список |
| ИНН | Реквизиты юрлица (информационно) |
| 💬 «Чат с клиентом» | Перейти в чат с клиентом из заказа |

### Дата и время доставки

| Элемент | Действие |
|---|---|
| «‹» / «›» | Переключать дату по дням |
| Поле даты | Открыть календарь |
| «✕» | Очистить дату |
| 🕐 | Выбрать временной интервал |
| «▾» | Выпадающий список слотов |

### Адрес доставки

- «∨» — выбрать другой адрес из адресов клиента.
- 📍 — открыть адрес на карте.

### Корзина

| Элемент | Действие |
|---|---|
| Название товара + ✏️ | Изменить цену вручную |
| 🗑️ | Удалить товар из заказа |
| – / + или вручную | Изменить количество |
| Доставка + ✏️ | Изменить стоимость доставки вручную |
| «+ Добавить товар» | Добавить товар через меню каталога |
| Бутыли к возврату + ✏️ | Изменить кол-во бутылей к возврату |
| Итого / К оплате / Баллами | Информационные строки с суммами |

### Водитель

«🚗 Водитель ▾» — назначить / сменить водителя.

### Вкладки истории (5 штук)

| Вкладка | Содержимое |
|---|---|
| Вся история | Полная хронология всех изменений |
| Комментарии | Комментарии операторов |
| Оператор | Действия операторов CRM |
| Клиент | Действия клиента (приложение / сайт) |
| Курьер | Действия водителя по заказу |

**Типы событий в ленте:** списание/начисление баллов, смена статуса (кем), смена способа оплаты/типа/суммы, изменение интервала, создание/добавление товара, установка адреса. Синие ссылки → связанный объект. Пагинация ленты: «‹ 1 2 ›».

## 5. Форма «Создание заказа»

Открывается: кнопка «+ Новый заказ» на странице заказов или «+» в карточке клиента.

### Поиск клиента

| Элемент | Действие |
|---|---|
| Выпадающий список поля ▾ | Везде / Дом-кв / Адрес / ФИО / Телефон / ID / Email / Организация |
| Поле «Поиск» + 🔍 | Найти клиента по выбранному полю |
| Кнопка «+» | Создать нового клиента прямо внутри формы |

### Тип, дата, интервал, оплата

- «Доставка» / «Самовывоз» — переключатель типа заказа.
- «‹» / дата / «✕» / «›» — навигация по датам.
- Временные слоты — клик для выбора интервала.
- Радиокнопки оплаты: QR code / Проверочный / По счёту (юрлица) / Наличка.
- Поле «Комментарий к заказу» — для водителя / оператора.

### Каталог товаров

- Вкладки категорий: Вода / Услуги / Напитки / Штучные товары / Акции / Оборудование / Комбо / «...» (скрытые).
- Строка товара: изображение, название, цена, остаток / мин. / макс. кол-во.
- «🛒 Добавить» — добавить в корзину.

### Корзина

| Элемент | Действие |
|---|---|
| «–» / «+» | Изменить количество |
| ✏️ рядом с ценой | Изменить цену вручную |
| «Стоимость доставки» + ✏️ | Изменить вручную |
| Долг по таре | Информация о долге |
| «–» / «+» / вручную бутылей | Задать кол-во бутылей к возврату |
| «Оформить [сумма] ₽» | Создать и сохранить заказ |
| «✕» | Закрыть форму без сохранения |

### Краткая информация о клиенте (правая панель)

- Организация (если есть), ФИО клиента, Адрес (можно изменить / добавить новый), Телефон (выпадающий список), Комментарий.
- Статус / дата / номер последнего заказа + возможность повторить.
- Краткая информация о последнем заказе + возможность повторить его.
- Баллы клиента.
- «Очистить ✕» — выбрать другого клиента.

**Контекстное меню МЛ (⋯):** печать PDF, экспорт CSV, очистить и удалить МЛ.
