# yandex-tracker-issue-creator

Создаёт задачи в Yandex Tracker в диалоговом режиме. Уточняет заголовок и описание, запрашивает список очередей и исполнителей, показывает превью и создаёт задачу после подтверждения.

## Как работает

1. Уточняет заголовок и описание задачи (если недостаточно данных)
2. Формирует чёткий заголовок (глагол + объект, до 80 символов) и структурированное описание
3. Получает полный список очередей через `queues_get_all` → пользователь выбирает нужную
4. Получает список пользователей через `users_get_all` → пользователь выбирает исполнителя
5. Показывает превью задачи и ждёт подтверждения
6. Проверяет обязательные поля очереди через `queue_get_fields`
7. Создаёт задачу через `issue_create`

**Шаги 3 и 4 обязательны — пропускать нельзя.**

## Входные данные

Любое описание задачи в свободной форме:

- «Создай задачу: исправить баг авторизации»
- «Занеси в трекер: добавить экспорт в Excel»
- Список фич, улучшений, багов

## Выходные данные

```
✅ QUEUE-123 — Исправить ошибку авторизации
Очередь: Backend | Исполнитель: Иван Иванов
```

Готовая задача в Yandex Tracker с присвоенным номером.

## Триггеры

«Создай задачу», «добавь задачи в трекер», «занеси в яндекс трекер», «поставь таски», «создай issues», «заведи задачи», упоминание очереди + создание.
