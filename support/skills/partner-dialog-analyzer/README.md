# partner-dialog-analyzer

Анализирует диалоги с партнёрами (чаты и транскрибации звонков) по двум осям: соответствие тону общения (TOV) и работа с задачами. Автоматически оценивает качество коммуникации поддержки и выявляет потерянные договорённости.

## Как работает

1. Определяет тип входных данных (чат или транскрибация звонка)
2. Анализирует TOV по 6 параметрам с цитатами из диалога
3. Извлекает все задачи — явные и неявные — и классифицирует их по статусу
4. Генерирует Word-документ с тремя разделами: TOV, задачи, рекомендации
5. Отдаёт файл через `present_files`

## Входные данные

- Файлы: `.txt`, `.md`, `.docx`, `.pdf`
- Текст диалога прямо в чате
- Поддерживаются чаты (переписка) и транскрибации звонков

## Выходные данные

Word-документ (`.docx`) с тремя разделами:

| Раздел | Содержание |
|--------|-----------|
| Оценка TOV | 6 параметров с цитатами и оценкой ✅/⚠️/❌ |
| Таблица задач | Все задачи: решена / открыта / потеряна / не в трекере. Цветовая кодировка строк |
| Рекомендации | Что сделать сейчас, что занести в Yandex Tracker |

## Эталон TOV

Оценка производится по стандарту Aqua Delivery v1.0 (2025). 5 характеристик голоса: конкретика, дружелюбность без панибратства, уверенность без снисходительности, ответственность за результат, завершённость каждого обращения.

## Триггеры

«Проверь диалог», «разбери переписку», «анализ звонка», «что решили», «какие задачи», «TOV», «чат с партнёром», загрузка txt/docx/pdf с диалогом.
