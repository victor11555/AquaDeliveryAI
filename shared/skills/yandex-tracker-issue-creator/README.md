# yandex-tracker-issue-creator

Создаёт задачи в Yandex Tracker в диалоговом режиме.

## Как вызвать

`/yandex-tracker-issue-creator`

## Триггеры

«создай задачу», «занеси в трекер», «поставь таски», «добавь в очередь», «заведи задачи»

## Что делает

1. Уточняет заголовок и описание задачи
2. Получает список очередей через `queues_get_all` — показывает пользователю, ждёт выбора
3. Получает список пользователей через `users_get_all` — показывает, ждёт выбора исполнителя
4. Показывает превью задачи, ждёт подтверждения
5. Создаёт задачу через `issue_create`, возвращает номер

## Входные данные

Любое описание задачи в свободной форме.

## Выходные данные

```
✅ QUEUE-123 — Заголовок задачи
Очередь: Название | Исполнитель: Иван Иванов
```
