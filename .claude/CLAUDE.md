# AquaDeliveryAI — Project Context

Репозиторий скиллов для платформы Aqua Delivery. Каждый отдел компании имеет свою папку со скиллами — инструкциями для AI-агентов, автоматизирующих рабочие процессы.

---

## Структура репозитория

```
AquaDeliveryAI/
├── .claude/
│   ├── skills/          ← копии скиллов для вызова через /skill-name
│   └── settings.json
├── support/             ← техподдержка партнёров
├── product/             ← продуктовый отдел
├── 1c-implementation/   ← внедрение 1С УНФ
├── marketing/           ← маркетинг (скиллов пока нет)
├── sales/               ← продажи (скиллов пока нет)
├── accounting/          ← бухгалтерия (скиллов пока нет)
├── testing/             ← тестирование (скиллов пока нет)
├── CLAUDE.md            ← этот файл
├── MEMORY.md            ← индекс памяти проекта
├── project_skill_structure.md
└── readme_rules.md
```

## Правила структуры скиллов

Каждый скилл живёт в собственной папке:

```
<отдел>/skills/<skill-name>/
├── <skill-name>.md     ← основной файл скилла (алгоритм)
├── README.md           ← описание, входные/выходные данные, структура
├── references/         ← только если есть референсные файлы
└── scripts/            ← только если есть скрипты
```

**Папки `references/` и `scripts/` создавать только если они реально нужны.**

При создании скилла также скопировать `<skill-name>.md` в `.claude/skills/` для вызова через `/skill-name`.

---

## Правила README

В репозитории два уровня README — оба всегда актуальны:

**README отдела** (`<отдел>/README.md`):
- Одна строка: что делает отдел
- Таблица всех скиллов с описаниями
- Файловая структура отдела
- Обновлять при добавлении / удалении / переименовании скилла

**README скилла** (`<отдел>/skills/<skill-name>/README.md`):
- Что это, как работает, входные/выходные данные, триггеры, файловая структура
- Обновлять при изменении алгоритма или файлов скилла

## Чеклист при добавлении нового скилла

1. Создать папку `<отдел>/skills/<skill-name>/`
2. Положить `<skill-name>.md` (основной файл скилла)
3. Создать `README.md` рядом со скиллом
4. Обновить `<отдел>/README.md` — добавить строку в таблицу и в файловую структуру
5. Скопировать `<skill-name>.md` в `.claude/skills/`
6. Закоммитить и запушить
