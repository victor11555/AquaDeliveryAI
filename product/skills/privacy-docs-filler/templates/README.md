# Шаблоны документов

Сюда необходимо положить три `.docx` файла-шаблона:

- `Политика конфиденциальности.docx`
- `Согласие на обработку персональных данных.docx`
- `Пользовательское соглашение о порядке покупки товаров.docx`

Скрипт `scripts/fill_docs.py` читает эти файлы через `zipfile` (XML внутри docx)
и заменяет плейсхолдеры реквизитами организации.

Исходное содержимое шаблонов хранится рядом в виде `.md` файлов для справки.
