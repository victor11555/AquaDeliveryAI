# Product

Продуктовый отдел. Скиллы автоматизируют подготовку юридических и продуктовых документов для партнёров платформы Aqua Delivery.

## Скиллы

| Скилл | Описание |
|-------|----------|
| [privacy-docs-filler](skills/privacy-docs-filler/) | Заполняет юридические документы (Политика конфиденциальности, Согласие на обработку ПД, Пользовательское соглашение) реквизитами организации. Извлекает данные из PDF, подставляет в шаблоны и генерирует HTML-файлы для печати. |

## Файловая структура

```
product/
├── README.md
└── skills/
    └── privacy-docs-filler/
        ├── privacy-docs-filler.md
        ├── README.md
        ├── scripts/
        │   ├── fill_docs.py
        │   └── style_html.py
        └── templates/
            ├── Политика конфиденциальности.docx
            ├── Согласие на обработку персональных данных.docx
            └── Пользовательское соглашение о порядке покупки товаров.docx
```
